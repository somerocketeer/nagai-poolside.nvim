*nagai-poolside.txt* Nagai Poolside                                    *nagai-poolside*

INTRODUCTION                                                     *nagai-poolside-intro*
    Nagai Poolside is a dusk-inspired palette that pairs deep teal bases
    with bright pool-blue highlights. This help file documents configuration
    options and integrations exposed by the plugin.

SETUP                                                           *nagai-poolside-setup*
    Lua (Lazy.nvim) sample:
>
    require('nagai_poolside').setup({ transparent = false })
    vim.cmd.colorscheme('nagai-poolside')
<

OPTIONS                                                         *nagai-poolside-options*
    transparent    (boolean)  When true, leaves main window backgrounds as
                              `NONE` so terminal/parent UI shows through.
    user_overrides (function) Optional callback invoked after highlights
                              are applied. Use to tweak highlight groups
                              without forking the colorscheme.

EXTRAS                                                          *nagai-poolside-extras*
    Lualine integration:
>
    require('lualine').setup({
      options = { theme = 'nagai-poolside' },
    })
<

    Mini.statusline receives custom mode colors automatically when the
    colorscheme is loaded.
